# <a name="ef-core-tools--extensions"></a>Ferramentas e Extensões do EF Core

Ferramentas e extensões oferecem funcionalidade adicional para o Entity Framework Core.

As extensões são criadas por uma variedade de fontes. Nem todas as extensões são mantidas como parte do projeto do Entity Framework Core. Ao considerar uma extensão de terceiros, avalie a qualidade, o licenciamento, o suporte etc. para garantir ela cumpra os requisitos.
